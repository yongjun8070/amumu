# 심해 탐험 3D (Deep Sea Explorer 3D) 🌊

**완전한 3D 1인칭 시점**으로 심해를 탐험하는 몰입감 넘치는 브라우저 게임입니다.

Three.js를 사용하여 제작된 이 게임은 마우스로 360도 자유롭게 둘러보며, 실제 잠수부가 된 것처럼 심해의 신비를 탐험할 수 있습니다!

## 🎮 게임 특징

### 🌟 진짜 3D 경험
- **1인칭 시점**: 잠수부의 눈으로 직접 보는 시점
- **마우스 360도 회전**: 마우스를 움직여 모든 방향을 자유롭게 둘러보기
- **완전한 3D 이동**: 전후좌우 + 상하 이동으로 3차원 공간 탐험
- **실시간 조명**: 손전등 효과로 어두운 심해를 밝히기
- **동적 안개 효과**: 깊이에 따라 변화하는 가시거리

### 🐟 다양한 3D 생물들
- **10종 이상의 3D 모델**: 각 생물마다 고유한 3D 형태
- **리얼한 애니메이션**: 유영, 촉수 움직임, 회전 등
- **발광 효과**: 심해 생물들의 생체 발광
- **거대 생물**: 최대 25미터 크기의 압도적인 고대 생물
- **AI 행동**: 큰 생물은 플레이어를 쫓아옴

### 🏛️ 3D 유적과 환경
- **고대 유적**: 깊은 곳에서 발견되는 3D 석상과 기둥
- **빛나는 조명**: 유적에서 나오는 신비한 청록색 빛
- **해양 눈 파티클**: 2000개의 파티클로 표현되는 해양 눈
- **동적 배경**: 5개 깊이 존마다 다른 색상과 분위기

## 🎯 조작법

### 기본 조작
- **마우스 이동**: 360도 시점 회전 (상하좌우 자유롭게)
- **W**: 앞으로 이동
- **S**: 뒤로 이동
- **A**: 왼쪽으로 이동
- **D**: 오른쪽으로 이동
- **SPACE**: 상승
- **SHIFT**: 하강
- **L**: 조명 켜기/끄기
- **ESC**: 수면으로 긴급 귀환

### 게임 시작
1. "탐험 시작" 버튼 클릭
2. 화면 클릭하여 Pointer Lock 활성화 (마우스 잠금)
3. 마우스로 둘러보고 키보드로 이동!

## 🌊 깊이 존

### 1. 표층 - Epipelagic Zone (0-200m)
- 밝고 평화로운 구역
- 물고기, 해파리
- 가시거리: 매우 좋음

### 2. 중층 - Mesopelagic Zone (200-1,000m)
- 빛이 약해지는 황혼대
- 오징어, 아귀
- 가시거리: 보통

### 3. 심층 - Bathypelagic Zone (1,000-4,000m)
- 완전한 어둠
- 바이퍼피쉬, 발광 생물
- 가시거리: 나쁨

### 4. 심해층 - Abyssopelagic Zone (4,000-6,000m)
- 극한의 압력
- 대왕심해등각류, 뱀파이어오징어
- 가시거리: 매우 나쁨

### 5. 초심해층 - Hadal Zone (6,000m+)
- 알려지지 않은 세계
- **심연의 뱀** (10m)
- **리바이어던** (20m)
- **고대의 존재** (25m)
- **고대 유적 출현**
- 가시거리: 극악

## 📊 게임 시스템

### 자원 관리
- **산소**: 지속적으로 감소, 수면에서 회복
- **조명**: 켜져 있을 때만 배터리 소모
- **압력**: 깊이에 비례하여 증가

### 생물 발견
- 생물 근처(20m 이내)로 가면 자동 발견
- 발견한 생물 종류가 HUD에 표시됨
- 최대한 많은 생물을 발견하는 것이 목표!

### 생존 팁
1. **산소 관리가 핵심**: 깊이 내려갈수록 산소 소모가 빠름
2. **조명 절약**: 꼭 필요할 때만 켜기
3. **거대 생물 조심**: 7000m 이상에서는 리바이어던 조심!
4. **수면 귀환**: ESC 키로 빠르게 수면으로 탈출
5. **천천히 탐험**: 급하게 내려가지 말고 각 깊이를 즐기세요

## 🚀 실행 방법

### 간단한 방법 (권장)
1. `index.html` 파일을 브라우저에서 열기
2. "탐험 시작" 클릭!

### 로컬 서버로 실행
```bash
# Python 3
cd deep-sea-explorer
python3 -m http.server 8000

# 또는 Node.js
npx http-server -p 8000
```

그리고 브라우저에서 `http://localhost:8000` 접속

## 💻 기술 스택

- **Three.js r128**: 3D 렌더링 엔진
- **WebGL**: 하드웨어 가속 그래픽
- **Pointer Lock API**: 1인칭 마우스 컨트롤
- **Vanilla JavaScript**: 순수 자바스크립트
- **HTML5 Canvas**: 렌더링 대상

## 🎨 3D 모델 특징

### 생물별 3D 디자인
- **해파리**: 반투명 돔 + 8개 촉수
- **물고기**: 타원형 몸체 + 꼬리 지느러미
- **아귀**: 원뿔형 몸 + 발광 미끼 + 날카로운 이빨
- **거대 생물**: 구형 몸체 + 다수의 빛나는 눈 + 12개 촉수

### 조명 시스템
- **주변광**: 깊이에 따라 자동 조절
- **손전등**: SpotLight로 구현된 방향성 조명
- **생물 발광**: PointLight로 구현
- **유적 조명**: 청록색 발광 효과

### 파티클 효과
- **해양 눈**: 2000개 파티클, 플레이어 주변 순환
- **깊이 200m 이상**에서만 표시
- 실시간 위치 업데이트

## 🎯 성능 최적화

- **LOD (Level of Detail)**: 거리에 따른 디테일 조절
- **Frustum Culling**: 화면 밖 객체 렌더링 스킵
- **생물 수 제한**: 각 종류당 최대 2마리
- **동적 생성/제거**: 플레이어 주변만 활성화

## 🐛 알려진 이슈

- 일부 브라우저에서 Pointer Lock이 작동하지 않을 수 있음
- 모바일 기기는 지원하지 않음 (데스크톱 전용)
- 낮은 사양 PC에서는 프레임 드롭 가능

## 📝 조작 팁

### 효율적인 탐험
1. 마우스를 천천히 움직여 주변을 관찰
2. W+Space로 대각선 상승하며 이동
3. 조명을 끄고 생물의 발광을 관찰
4. 거대 생물은 거리를 두고 관찰

### 생존 전략
1. 깊이 1000m씩 단계적으로 하강
2. 각 깊이에서 충분히 탐험 후 다음 단계로
3. 산소 50% 이하일 때 수면 복귀 고려
4. 조명은 생물 발견 시에만 사용
